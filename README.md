## Usage

1. Install dependencies

```
yarn
```

1. Generate source data

```
yarn gen-data
```

3. Run the application
   
```
yarn start
```

and open http://localhost:3000 with your browser to see the result.

## Development

Run 

```
yarn dev
```

## Deploy

This website is automatically deployed on vercel every day at midnight.
